# heypath
NASのファイルパスをMac形式もしくはWindows形式に変換します。  

## Development Environment
Node.js v14.18.0 

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
